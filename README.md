## How to Start

**Node.js version 18 or higher is required.**

### Yarn Berry 
```bash
npm install yarn -g
yarn set version berry
yarn
```

### Project Run (Development)
```bash
yarn dev
```

### Project Run (Production)
```bash
yarn build
yarn start
```
